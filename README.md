Historical import reconstructed from the old projects index.

## Details
- Year: 1995
- Environment: NextStep
- Status: Finished

## Description
JollysClass2 is a Fax Class 2 nextstep system driver. In case you got a modem that supports the fax class 2 standard you can fax from your nextstep/openstep system now.

Even though the README states that this driver is shareware - it's now for free.
